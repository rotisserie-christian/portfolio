### /starfield/utils 
- `starConfig.ts` - Default configurations
- `starTypes.ts` - PropTypes and default props
- `starMath.ts` - Math utilities (distortion, swirl calculations)
- `starRendering.ts` - Functions for drawing stars, and stars with gravity distortion trails 
- `shootingStarMath.ts` - Math utilities (shooting stars spawn)
- `shootingStarRendering.ts` - Get the props needed to render a shooting star
