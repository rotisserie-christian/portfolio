### /starfield/utils 
- `starConfig.js` - Default configurations
- `starTypes.js` - PropTypes and default props
- `starMath.js` - Math utilities (distortion, swirl calculations)
- `starRendering.js` - Functions for drawing stars, and stars with gravity distortion trails 
- `shootingStarMath.js` - Math utilities (shooting stars spawn)
- `shootingStarRendering.js` - Get the props needed to render a shooting star
