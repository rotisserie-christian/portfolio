## Crayonbrain
- **`/hooks`** - Logic for sequencing audio and rendering visuals 
- **`/ui`** - Drum pad and tempo/preset/controller componenets 
- **`/utils`** - Constants, utilities, and error classes
- **`Crayonbrain.jsx`** - Main container 
- **`DemoSequencer.jsx`** - Drum sequencer
- **`Visualizer.jsx`** - renders the visualizer canvas and delegates to useVisualizer
