## Crayonbrain
- **`/hooks`** - Logic for sequencing audio and rendering visuals 
- **`/ui`** - Drum pad, features, and controller components 
- **`/utils`** - Constants, utilities, and error classes
- **`Crayonbrain.jsx`** - Main container with lazy loading coordination
- **`DemoSequencer.jsx`** - Shell component that toggles between features and demo mode
- **`SequencerInner.jsx`** - Gated core containing Tone.js logic and asset imports
- **`Visualizer.jsx`** - Gated visualizer canvas container
