### Utilities
- **`analyserSetup.js`** - Creates and configures Web Audio analyser nodes
- **`cn.js`** - Merge Tailwind CSS classes using clsx and tailwind-merge. Enables tailwind conditionals and removes duplicate classNames
- **`presetLoader.js`** - Loads Butterchurn presets 
- **`visualizerConstants.js`** - Configuration constants for visualizer
